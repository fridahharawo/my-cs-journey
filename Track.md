## 📍 Why This Path?

Kenya's tech sector is growing at **23% annually** (Silicon Savannah). The most in-demand skills right now:
- Python & JavaScript (software development)
- AI/ML tools and API integration
- Cloud basics (AWS / Google Cloud)
- Networking fundamentals (for DevOps + Security roles)
- Cybersecurity (10,000+ expert shortage by end of 2026)

Entry-level software roles pay **KES 50,000–200,000/month**. AI-skilled roles command a **56% wage premium** over non-AI peers. Your path targets this gap.

## 📦 PHASE 1 — Foundations (Month 1–2)

### Level 1: Networking Fundamentals
**Duration:** 2 weeks (≈28 hours)  
**Status:** 🔲 Not Started

#### What You'll Learn
- [x] How the internet works (IP, DNS, HTTP/HTTPS)
- [ ] OSI Model — the 7 layers, in plain English
- [ ] TCP vs UDP — when each is used
- [ ] Subnetting basics (IP address ranges)
- [ ] What a router, switch, and firewall do
- [ ] How a web request travels from browser to server

#### Resources (All Free)
- [Cisco Networking Basics — free NetAcad course](https://www.netacad.com/courses/networking-basics)
- [Professor Messer's CompTIA Network+ Notes](https://www.professormesser.com/network-plus/n10-008/n10-008-video/n10-008-training-course/)
- YouTube: "NetworkChuck" channel — very beginner friendly

#### Practice
- Use `ping`, `traceroute`, `nslookup` in your terminal
- Draw the OSI model from memory — label what happens at each layer

#### Milestone Test ✅
> Before moving to Level 2, you must be able to:
> 1. Explain what happens step-by-step when you type "google.com" in a browser
> 2. Name all 7 OSI layers and give one example of a protocol at each
> 3. Explain the difference between TCP and UDP with a real-world analogy
> 4. Read an IP address and identify its network vs host portion

**Milestone Status:** 🔲 Pending

## 📦 PHASE 5 — Internship Prep (Month 9–10)

### Level 9: Portfolio + CV + LinkedIn
**Status:** 🔲 Not Started

#### What You'll Build
- [ ] A polished portfolio site with 4–5 real projects
- [ ] A 1-page CV tailored for tech roles in Kenya
- [ ] A LinkedIn profile with: About, Experience, Projects, Skills, Certifications

#### CV Must-Haves
- Name, contacts, GitHub link, deployed portfolio link
- Projects (with links and 2-sentence descriptions each)
- Skills section: Python, JavaScript, SQL, REST APIs, Git, AWS basics
- Education + any certifications
- No fluff — every line must prove a skill

#### Where to Apply for Internships (Kenya)
- **Andela** — andela.com (trains and places African devs)
- **Safaricom DigiFarm / Safaricom Internship** — careers.safaricom.co.ke
- **Microsoft Africa Development Centre** — careers.microsoft.com (competitive)
- **Google Kenya** — careers.google.com
- **iHub Nairobi** — ihub.co.ke (startup ecosystem)
- **Moringa School** — moringaschool.com (sometimes hiring TAs)
- **Zindi Africa** — zindi.africa (competitions + job board)
- **LinkedIn Jobs** — filter: Nairobi, Internship, Software/AI
- **Brighter Monday** — brightermonday.co.ke
- **MyJobMag Kenya** — myjobmag.co.ke

#### How to Handle Internship Applications
1. **Tailor your CV per role** — match keywords from job description
2. **Cold email is real** — find the hiring manager on LinkedIn, send a 3-sentence message
3. **Attach your GitHub + portfolio link in every application**
4. **Prepare 3 project stories** using STAR format (Situation, Task, Action, Result)
5. **Know your projects deeply** — be ready to demo them live in interviews

#### Interview Prep Topics
- [ ] Python basics quiz (data structures, OOP basics)
- [ ] SQL queries (JOINs, GROUP BY, subqueries)
- [ ] System design basics (explain how you'd build a simple app)
- [ ] Behavioral questions: "Tell me about a challenge you solved"
